# cafeNext
Naver Cafe Simple Clone

## Framework
* config - Spring Boot
* entity - JPA
* repository - Spring Data JPA
* test - spock with spring
* web view - Velocity, FreeMarker
* javascript - jQuery

## Structure
* cafe-core
** Cafe Commone Component
** Entity, Repository, Service
* cafe-cafeweb
** Individual Cafe
** WebMVC
* cafe-sectionweb
** Cafe Section
** WebMVC
* cafe-api
** internal API
** Rest API
